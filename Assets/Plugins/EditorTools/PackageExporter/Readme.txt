# HOW TO OPEN

In the menu bar: [Tools] -> [Package Exporter]
or press F4

# HOW TO USE

Drag the folder you want to package in the "Folder" field, set up all the info and hit the "Export Package" button, the package will be exported in the Builds folder inside your project folder.

Note that this tool only packs the stuff inside the selected folder.
